- TypeScript のインストール

```
$ npm install -g typescript
```

- tsconfic.json の雛形作成

```
$ tsc --init
```

- tsc コマンドの実行

```
$ tsc
```

- カレントディレクトリの tsconfig.json 参照

```
$ tsc -b
```

- src/tsconfig.json 参照

```
$ tsc -b src
```

- src/tsconfig.client.json 参照

```
$ tsc -b src/tsconfig.client.json
```
